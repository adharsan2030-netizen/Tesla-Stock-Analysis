# Tesla Stock Analysis Dashboard (Power BI)

## 📊 Overview
This project is a Power BI dashboard analyzing Tesla stock data.

## 🔧 Tools Used
- Power BI
- DAX
- Excel/CSV dataset

## 📈 Features
- Stock price trends over time
- Volume analysis
- Key performance metrics
